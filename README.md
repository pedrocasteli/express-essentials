# Express Essentials

This project was bulit by following along to the [Express Essentials] course on [LinkedIn Learning].

## Install and Run

This project requires [Node.js] to run. My version at the time was **20.15.0**.

1. Install the necessary dependencies:

```sh
npm install
```

2. Start the server:

```sh
npm start
```
